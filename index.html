bitnoise frontend sandbox v0.1

What's inside?

  • Basic html template
  • Basic UI directories setup (sass, js)
  • Bower config with jQuery, modernizr, html5shiv
  • Grunt config with compass, watch, jshint, cssmin, uglify etc.
  • Grunt tasks like watch or dump
  • Basic setup of the sass files (base, typo, mixins etc.)
  • Basic setup of js (App object with init)

Install

  • git clone git@github.com:Bitnoise/BtnFrontendSandbox.git
  • npm install
  • grunt install
  • grunt watch
